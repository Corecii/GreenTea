Type

Represents a type we can check against or inspect.
Use the methods to check against the type.
Use the properties to inspect the type.

Cause​

</>
type Cause = {
ok: boolean,
encompassingType: Type?,
errs: {{
type: any,
input: any?,
message: string?
}},
formatErr: (self: Cause) → string

Represents the result of a :matches() call.\
Includes a __tostring metamethod, which formats to a readable error.

type​

</>
Type.type(self: any) → ...any

In type definitions via typeof(Type.type()), returns the Luau type this GreenTea type represents.\
At runtime, returns self. This allows you to compose complex GreenTea types by passing GreenTea.type() into a GreenTea constructor. (Only applies when "built" with GreenTea.build(GreenTea constructors here))
